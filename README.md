### 🏥 Click Clínicas  
Sistema SaaS para clínicas, com gestão de pacientes, agenda, financeiro, automações, documentos, fotos, orçamentos e recursos voltados para organização da operação clínica.

### 📅 Agenda365  
Plataforma de agendamentos online, criada para facilitar marcações, confirmações, lembretes e organização de horários para negócios e profissionais.

### 🎟️ Rifa365
